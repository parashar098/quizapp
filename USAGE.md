# QuizMaster - Quick Start Guide

## Getting Started

### Teacher Flow

1. **Sign Up**
   - Click "Sign Up" button on the landing page
   - Enter your name, email, and password
   - Select "Teacher" role
   - Click "Sign Up"

2. **Dashboard Overview**
   After logging in, you'll see:
   - Total Quizzes created
   - Total Students who attempted your quizzes
   - Total Attempts across all quizzes

3. **Create Your First Quiz**
   - Click "Create Quiz" in the sidebar
   - Fill in:
     - Quiz Title (e.g., "JavaScript Basics")
     - Description (e.g., "Test your JavaScript knowledge")
     - Duration in minutes (e.g., 30)
   - Add questions:
     - Type your question
     - Enter 4 options
     - Click the circle next to the correct answer
     - Click "Add Question"
   - Repeat for all questions
   - Click "Create Quiz" when done

4. **Share Quiz with Students**
   - Go to "My Quizzes" tab
   - Find the join code displayed on each quiz card
   - Share this code with your students

5. **View Results**
   - Click the eye icon on any quiz card
   - See all students who attempted the quiz
   - View their scores

### Student Flow

1. **Sign Up**
   - Click "Sign Up" button on the landing page
   - Enter your name, email, and password
   - Select "Student" role
   - Click "Sign Up"

2. **Dashboard Overview**
   After logging in, you'll see:
   - Quizzes Attempted
   - Average Score
   - Best Score

3. **Browse Quizzes**
   - Click "Available Quizzes" in the sidebar
   - See all quizzes created by teachers
   - Each quiz shows:
     - Title and description
     - Duration
     - Number of questions
     - Join code
     - Teacher name

4. **Take a Quiz**
   - Click "Start Quiz" button
   - The timer starts automatically
   - For each question:
     - Read the question carefully
     - Click on your answer choice
     - Use "Next" to move to the next question
     - Use "Previous" to go back
   - Click "Submit Quiz" on the last question

5. **View Results**
   - Immediately after submitting, see:
     - Your score (e.g., 8/10)
     - Percentage score
     - Answer review showing:
       - Each question
       - Your answer (marked in red if wrong)
       - Correct answer (marked in green)
   - View all past results in "My Results" tab

## Key Features Explained

### Timer
- Appears in the top-right corner during quiz
- Shows remaining time
- Turns red when less than 20% time remaining
- Auto-submits quiz when time runs out

### Quiz Join Codes
- 6-character alphanumeric code
- Unique for each quiz
- Easy for students to reference
- Displayed on quiz cards

### Answer Review
- Shows all questions after quiz submission
- Green checkmark for correct answers
- Red X for incorrect answers
- Displays both your answer and correct answer

### One Attempt Per Quiz
- Students can only attempt each quiz once
- Prevents multiple attempts to game the system
- Fair assessment for teachers

## Tips for Teachers

1. **Quiz Duration**: Consider 1-2 minutes per question
2. **Question Quality**: Make sure options are clear and unambiguous
3. **Preview**: Create a test quiz first to see the student experience
4. **Join Codes**: Write them on the board or share via email/chat

## Tips for Students

1. **Read Carefully**: Read each question thoroughly
2. **Time Management**: Keep an eye on the timer
3. **Review First**: Use Previous/Next to review answers before submitting
4. **No Rush**: Take your time but be mindful of the countdown
5. **Answer All**: Make sure you've answered every question

## Troubleshooting

### Can't Log In?
- Check your email and password
- Ensure you signed up with the correct role
- Try the "Forgot Password" option (if implemented)

### Quiz Not Showing?
- Make sure you're in the "Available Quizzes" tab
- Refresh the page
- Check if you've already attempted it

### Timer Issues?
- Don't refresh the page during a quiz
- Ensure stable internet connection

### Can't Submit Quiz?
- Make sure you've answered all questions
- Check if there's time remaining
- Click "Submit Quiz" button, not browser back button

## Best Practices

### For Teachers
- Create quizzes with 5-15 questions for optimal engagement
- Use clear, concise question text
- Avoid trick questions
- Provide reasonable time limits
- Review results regularly

### For Students
- Take quizzes in a quiet environment
- Ensure stable internet connection
- Don't close or refresh the browser during a quiz
- Review your results to learn from mistakes
